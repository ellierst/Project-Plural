Задачу та всі її підпункти виконав: Мазур Владислав

У програмі використовується структура Person для представлення осіб та клас Set для зберігання імен людей, яких відвідувала кожна особа. Клас Set використовується для перевірки наявності певних імен у множинах візитів, а також для обчислення перетинів та різниць між множинами.

Програма виконує наступні основні завдання:

1. Люди, які ходять в гості найменше разів.
2. Люди, які відвідали усіх своїх друзів.
3. Пошук тісного кола з трьох або чотирьох друзів.
4. Вивід таблиці відвідувань для кожної особи.

Розглянемо виконання кожного завдання окремо:

Завдання 1: Люди, які ходять в гості найменше разів.
Програма починає з ініціалізації minVisits до максимально можливого значення (INT_MAX) та створення порожнього Set<string> для людей, які ходять в гості найменше разів (peopleWithMinVisits).
Програма проходить по масиву осіб (people) і для кожної особи (person) обчислює кількість відвідувань (visitCount), використовуючи метод getSize() об'єкта visits.
Якщо кількість відвідувань менша за поточний мінімум (minVisits), програма оновлює minVisits і створює новий Set для людей з найменшою кількістю візитів.
Якщо кількість відвідувань дорівнює мінімальній, програма додає ім'я людини до peopleWithMinVisits.
Наприкінці, програма виводить список людей, які ходять в гості найменше разів.

Завдання 2: Люди, які відвідали усіх своїх друзів
Програма проходить по кожній особі (person) з масиву people.
Для кожної особи, програма створює Set<string> (others) для зберігання імен усіх інших людей.
Потім програма перевіряє, чи person.visits містить усі імена з others, використовуючи метод intersect для обчислення перетину visits та others.
Якщо visits містить усі імена з others, програма виводить ім'я людини, яка відвідала всіх своїх друзів.

Завдання 3: Пошук тісного кола з трьох або чотирьох друзів
Програма використовує вкладені цикли для перевірки комбінацій трьох або чотирьох осіб.
Для кожної комбінації трьох осіб (people[i], people[j], people[k]), програма перевіряє, чи всі вони відвідували один одного, використовуючи методи contains та visits.
Якщо така комбінація знайдена, програма виводить, що було знайдено "тісне коло з трьох друзів".
Потім програма продовжує шукати "тісне коло з чотирьох друзів", перевіряючи комбінації з чотирьох осіб (people[i], people[j], people[k], people[l]).
Якщо знайдено таку комбінацію, програма виводить, що знайдено "тісне коло з чотирьох друзів".

Завдання 4: Вивід таблиці відвідувань для кожної особи
Програма проходить по кожній особі (person) у масиві people.
Для кожної особи програма виводить її ім'я, а потім викликає метод printVisits для виводу імен людей, яких вона відвідала.

Висновок:
Програма використовує класи Set та Person для представлення та обробки відвідувань людей. Завдяки вкладеним циклам і методам класу Set програма ефективно вирішує завдання та виводить результати у вигляді, який передбачено умовою.
